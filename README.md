# 🔧 FULLCOMP - Sitio Web Corporativo

## 💼 Descripción

FULLCOMP es un sitio web de una sola página diseñado para la empresa FULLCOMP INDUSTRIAL PRECISION, especialistas en ingeniería de aire comprimido y servicios de mantenimiento de compresores industriales. El sitio presenta una interfaz moderna y responsiva construida con HTML, CSS y JavaScript, inspirada en el diseño de Google Stitches y Tailwind CSS.

### ✅ Características Principales

- **📱 Diseño Responsivo**: Optimizado para dispositivos móviles, tablets y escritorio
- **🧭 Navegación Suave**: Scroll suave entre secciones con indicador de navegación activa
- **📋 Secciones Informativas**:
  - Servicios técnicos especializados
  - Marcas de compresores representadas
  - Galería de imágenes del equipo y trabajos
  - Información sobre rendimiento y mantenimiento
  - Ubicación con mapa integrado de Google
- **💬 Integración de Contacto**: Botón flotante de WhatsApp y formulario de contacto directo
- **🌙 Tema Oscuro/Claro**: Soporte para modo oscuro automático
- **♿ Accesibilidad**: Fuentes legibles y navegación intuitiva

### 💻 Tecnologías Utilizadas

- **HTML5**: Estructura semántica del sitio
- **CSS3**: Estilos personalizados con Tailwind CSS
- **JavaScript**: Interactividad y navegación dinámica
- **Google Fonts**: Inter y Space Grotesk para tipografía
- **Material Symbols**: Iconos consistentes
- **Google Maps**: Integración de mapa de ubicación

### 📦 Instalación y Uso

1. Clona el repositorio:
   ```bash
   git clone <url-del-repositorio>
   cd fullcomp
   ```

2. Abre `index.html` en tu navegador web preferido

### 📁 Estructura del Proyecto

```
fullcomp/
├── index.html          # Página principal
├── styles.css          # Estilos CSS (opcional, estilos inline en HTML)
├── script.js           # JavaScript para navegación
├── doc/
│   └── img/            # Imágenes del sitio
└── README.md           # Este archivo
```
